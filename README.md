# The DAFT model template

The DAFT model is a brief set of principles aimed to improve the reproducibility of research papers.
This repository contains a template with examples showing how to organize.

## Directories

- [`0-data`](0-data): Data files and datasets used in the study.
- [`1-analyses`](1-analyses): Scripts and outputs from individual analyses.
- [`2-figures`](2-figures): Source figure files for generating figure plates.
- [`3-text`](3-text): Source text files for final manuscript.
